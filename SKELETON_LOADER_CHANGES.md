# Skeleton-анимации для экрана плеера

## Добавленные файлы

### `ui/components/SkeletonLoaders.kt`
Файл содержит компоненты для skeleton-анимаций (заглушки загрузки):

- **`ShimmerBrush()`** - базовый shimmer-эффект с диагональной анимацией градиента
- **`TrackItemSkeleton()`** - скелет для элемента трека (обложка + текстовые блоки)
- **`CardSkeleton()`** - скелет для карточки (исполнители, альбомы, плейлисты)
- **`TrackListSkeleton()`** - список skeleton-элементов треков
- **`CardGridSkeleton()`** - сетка skeleton-карточек
- **`ShimmerBox()`** - универсальный shimmer-блок

## Изменения в `MainActivity.kt`

Заменён простой `CircularProgressIndicator` на контекстные skeleton-анимации для каждой вкладки:

- **Вкладки "Все треки" и "Избранное"** - вертикальный список skeleton-элементов треков
- **Вкладка "Артисты"** - сетка 2xN skeleton-карточек
- **Вкладка "Альбомы"** - сетка 3xN skeleton-карточек
- **Вкладка "Плейлисты"** - сетка 2xN skeleton-карточек

## Особенности реализации

1. **Адаптивный bottom padding** - skeleton учитывает состояние expandProgress для правильного отображения над навигацией
2. **Соответствие реальному контенту** - структура skeleton-элементов повторяет структуру реальных элементов UI
3. **Тематические цвета** - shimmer использует цвета из `AudioPlayerThemeExtended`
4. **Плавная анимация** - диагональный градиент с длительностью 1500ms
